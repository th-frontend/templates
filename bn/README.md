# BN Landing Page Template

This directory contains a static HTML and CSS example for a roofing landing page. The files demonstrate a hero banner, service cards and a call‑to‑action section.

## Files
- `index.html` – main page structure with placeholder text and images
- `style.css` – page styles including hero layout and service grid
- `images/` – sample images used in the template

## Usage
1. Open `index.html` in a browser to preview the template.
2. Replace the placeholder images in the `images/` folder and adjust the text in `index.html` for your company.
3. Modify colors, fonts and layout in `style.css` to match your branding.

The page is completely static and has no external dependencies.
